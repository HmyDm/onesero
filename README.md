## 使用方法：

1. **打包下载，然后安装node.js ( 地址如下 )** 

**[Node.js 中文网](https://nodejs.cn/)**

2. **安装依赖**

```html
在下载的本项目的目录下右击--在终端中打开
npm install    
```

然后运行即可

> **npm run dev**
